AutoHotkey Gids

Alles wat je nodig hebt om macro's te maken op je laptop

Windows

1 Snel starten

1
Rechtsklik op je bureaublad → Nieuw → AutoHotkey Script
2
Rechtsklik op het bestand → Edit Script (opent in Kladblok)
3
Schrijf je macro's, sla op, en dubbelklik het bestand om te starten
4
Je ziet een groen icoontje in de taakbalk — macro's zijn nu actief
5
Na aanpassingen: rechtsklik op taakbalk-icoontje → Reload Script
Elke regel die begint met ; is een commentaar en wordt genegeerd door AutoHotkey.

3 Basis macro's

; Tekst typen
#h::Send, Hallo, dit werkt!

; Popup berichtje tonen
#d::MsgBox, Dit is een popup!

; Programma openen
#c::Run, calc.exe
#n::Run, notepad.exe

; Tekst met Enter typen
#m::Send, Hallo, dit is een zin.{Enter}

; Ctrl+C simuleren (kopiëren)
#k::Send, ^c

; Klein berichtje zonder popup
#t::ToolTip, Script werkt!

4 Herhaling (Loop)

Exact aantal keer

#h::
Loop, 5
{
    Send, Hallo{Enter}
}
return

Zolang toets ingedrukt is

#h::
while GetKeyState("#h", "P")
{
    Send, Hallo{Enter}
    Sleep, 500  ; wacht 500ms tussen elke herhaling
}
return

Toggle (aan/uit)

#h::
toggle := !toggle
while toggle
{
    Send, Hallo{Enter}
    Sleep, 500
}
return
Sleep waardeBetekenis
Sleep, 1000.1 seconde (snel)
Sleep, 5000.5 seconde
Sleep, 10001 seconde
Sleep, 30003 seconden

5 Muis

; Muisklik op huidige positie
#c::Click

; Dubbelklik
#d::Click, 2

; Klik op specifieke positie (x, y)
#p::Click, 500, 300

6 Vensters

; Huidig venster minimaliseren
#m::WinMinimize, A

; Huidig venster maximaliseren
#x::WinMaximize, A

; Specifiek programma naar voren halen
#b::WinActivate, Chrome

7 Handige extra's

; Script pauzeren/hervatten met de Pause-toets
Pause::Suspend

; Script volledig afsluiten
#Escape::ExitApp
Zet #Escape::ExitApp altijd in je script — zo kun je het snel stoppen als een macro te snel gaat of blijft hangen.
